# Description
This proyect is a store simulation that uses react related technologies and other libraries to completely mock the user flow of an e-commerce site.

# Technologies in place:
## Front end
- React
- React Hooks
- React-router
- Redux
- Redux-persist
- Redux-saga
- Reselect
- Styled components
- SASS
## External services
- Firebase
- Heroku
- Stripe
